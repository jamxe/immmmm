    用了一周 Flomo ，简单来说是一个「哔言哔语」的 正经版 ，其特有的回顾功能有点新意，基于某个 TAG 或 批注某条信息，无压力积累碎片化的想法，说不准碎着碎着就成了深度思考！

    刚公布了 API 发布内容 ，iOS 快捷方式、Mac 的自动化和Alfred ，让发布内容更直接！

    PS：其续费页面有个教育优惠，非常直接的说明不做验证自由选择，点个赞！
