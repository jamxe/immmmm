https://lmm214.github.io/immmmm/media/js/jquery.min.js

改为：

https://cdn.jsdelivr.net/gh/lmm214/immmmm/media/js/jquery.min.js

其中 lmm214 和 immmmm 分别为 GitHub 的用户名和仓库名，之后的 /media/js/jquery.min.js 为静态文件地址。
